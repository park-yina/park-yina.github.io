# 커널 오브젝트와 Usage Count
여러번 강조한 내용이지만, 커널 오브젝트는 운영체제에게 종속적이기 때문에, 소멸시킬 권한 역시 운영체제에게 있다.<br>
## CloseHandler
프로세스가 생성되면 커널 오브젝트는 생성된다.<br>
그러나, 커널 오브젝트의 소멸은 프로세스의 소멸 시점과 항상 일치하지는 않는다.<br>
